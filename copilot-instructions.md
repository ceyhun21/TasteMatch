# TasteMatch Project Instructions

## Project Overview
TasteMatch ist eine innovative Restaurant-Discovery-App mit Tinder-ähnlicher Swipe-Funktion, Allergie-Filtering, Restaurant-Verifizierung und echten Nutzerbewertungen mit Fotos.

## ✅ Projektstruktur vollständig eingerichtet

### Frontend ✓
- React 18 + TypeScript + Vite
- Tailwind CSS für Styling
- Framer Motion für Animationen
- Zustand für State Management
- Firebase Authentication
- Komponenten: SwipeCard, RestaurantCard, ReviewList, AllergyFilter, AllergyInfo, MapCard

### Backend ✓
- Express.js API Server
- Firebase Admin SDK
- TypeScript Support
- Controller-basierte Architektur

### Features ✓
1. **Allergie-Filtering** - Multi-Select, 12 Allergen-Typen
2. **Swipe-Funktion** - Tinder-ähnliche Bedienung mit Framer Motion
3. **Google Maps Integration** - Entfernungsberechnung, Standorterkennung
4. **Restaurant-Verifizierung** - Email-basiert mit Codes
5. **Bewertungs-System** - 5-Sterne, Fotos, Allergie-Info
6. **User Reviews** - Mit Bildern und Allergie-Kommentaren

## Tech Stack
- **Frontend**: React 18, TypeScript, Tailwind CSS, Framer Motion, Zustand
- **Backend**: Node.js, Express, Firebase Admin SDK
- **Database**: Firebase Firestore
- **APIs**: Google Maps API, Firebase Cloud Storage
- **Authentication**: Firebase Auth

1. Firebase-Projekt erstellen
2. Google Maps API-Schlüssel abrufen
3. `.env` Dateien konfigurieren
4. `npm install` in frontend/ und backend/
5. Development Server starten: `npm run dev`
